## System Summary

TunnelSight AI is a Development Track system for intelligent 3D subsurface mapping and risk prediction. It brings together a live browser-based 3D inspection workspace, tunnel model visualisation, marker-based risk review, measurement, screenshot evidence capture and a defined AI architecture for reconstruction, hazard detection, risk scoring and expert-reviewed recommendations.
